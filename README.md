# rtsp-to-webrtc-client

Python client library for [RTSPtoWebRTC](https://github.com/deepch/RTSPtoWebRTC)

## Development

```
$ python3 -m venv venv
$ source venv/bin/activate
$ pip3 install -e .
$ pip3 install -r requirements.txt

# Running tests
$ pytest

# Formatting and linting
$ pre-commit run --all-files
```
